        5. HTML é uma linguagem utilizada principalmente para:
        Reposta: B) Estruturar páginas da web

        6. Qual é a tag correta para criar um título principal em HTML?
        Resposta: B) h1

        7. Qual tag é utilizada para criar um parágrafo?
        Resposta: B) p

        8.Qual tag é utilizada para inserir uma imagem?
        Resposta: A) img

        9. Qual atributo da tag img descreve a imagem para leitores
        de tela?
        Resposta: C) alt

        10. Qual tag é usada para criar uma lista não ordenada?
        Resposta: A) ul

        11. Qual tag define um item dentro de uma lista?
        Resposta: C) li

        12. Qual tag é utilizada para criar uma tabela em HTML?
        Resposta: B) table

        13.Qual elemento representa uma linha de uma tabela?
        Resposta: A) tr

        14.Qual é a declaração correta que define um documento HTML5?
        Resposta: B) DOCTYPE html

        15. Qual elemento define o título exibido na aba do navegador?
        Resposta: B) title

        16. Qual tag cria o maior nível de título?
        Resposta: A) h6

        17. O HTML é classificado como:
        Resposta: B) Linguagem de marcação

        18. Qual elemento agrupa imagem e legenda?
        Resposta: B) figure

        19. Qual tag define legenda de imagem?
        Resposta: B) figcaption

        20. Qual atributo define largura de imagem?
        Resposta: B) width
