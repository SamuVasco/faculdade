A tag div(divisão) no HTML é um contêiner genérico de nível de bloco, utilizado para agrupar elementos, estruturar layouts e aplicar estilos CSS ou manipulações JavaScript. Ela não possui significado semântico (não representa nada por si só) e, por padrão, ocupa toda a largura disponível, quebrando a linha antes e depois.
